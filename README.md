## Description

A Task Management backend using Nest.js

## Database setup

Install docker: [https://docs.docker.com/engine/install](https://docs.docker.com/engine/install)/

```bash
$ cd scailing-todo-backend
$ docker compose up -d
```

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
$ # development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
